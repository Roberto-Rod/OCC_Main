Note
To compile optimised version on the pc use "gcc -o pic32 main.c ndpa.c ndpa_tools.c -Dx86 -lm -Ofast [-std=c89]"
To compile with debug info on the pc use "gcc -o pic32 main.c ndpa.c ndpa_tools.c -Dx86 -lm -g -O0 -fno-inline -DDEBUG [-std=c89]"
To compile for the PIC32MZ load the project from NDPA.X
The PIC version of the code outputs (printf) to a serial port on a connected device using the mini usb port of the device. Connection: 115200 baud,8 bit,1 stop, No parity, XON/OFF
