A task to read the specified analog input channel(s) on-demand.

An initial sample is read immediately on task creation and then further samples on-demand if required until the task is stopped or cleared.

class framework.device.ni6363.DigitalInputOnChangeTask(lines: str, data: numpy.ndarray)

A task to read the specified digital input lines on-change.

An initial sample is read immediately on task creation using a Task and then further samples on-change until the task is stopped or cleared.

On-change detection requires buffered access so on the NI 6363 this class can only be used with port 0.

Inheritance
