class framework.module.serial_debug.SerialDebugLogger(device: str, baudrate: int, fname: str, mode: str, bufsize: int, timeout: float, queue: queue.Queue)

This class implements a serial debug logger.

Ultimately we might want the framework to be able to interact with the debug in some way e.g. waiting for a certain input but for now simply logging the debug output is sufficient.
